# Stage10 设计文档

> **Stage**：Stage10 — 回字形边距布局（Axis Margins Layout）
> **前置 Stage**：Stage5（AudioEQ 完整实现）
> **并行 SubStage**：10A 一个 SubStage
> **目标**：实现经典图表布局 — 内口（坐标轴+曲线+Band），外口（完整界面），文字标签在两口之间的边距区域。

---

## 背景与目标

当前渲染使用整个 Widget 区域作为坐标轴视口，导致：
1. 坐标轴刻度文字被裁剪或与边界重叠
2. 没有为文字标签预留空间
3. 布局不专业

### 目标布局（回字形）

```
┌──────────────────────────────────────────────────┐
│  ← top margin (10px)                             │
│  ┌──────────────────────────────────────────┐    │
│  │                                          │    │
│g │  Inner Chart Area（内口）                 │    │r
│a │  - Grid lines（频率+增益网格）            │    │i
│i │  - Curves（曲线）                         │    │g
│n │  - Band handles（圆形手柄）               │    │h
│  │  - LPF/HPF handles（椭圆手柄）            │    │t
│m │                                          │    │m
│a │                                          │    │a
│r │                                          │    │r
│g │                                          │    │g
│i │                                          │    │i
│n │                                          │    │n
│  └──────────────────────────────────────────┘    │
│  ← freq labels（频率标签：20 50 100 200 ...）     │
│     bottom margin (25px)                         │
└──────────────────────────────────────────────────┘
```

### 边距定义

| 方向 | 像素 | 用途 |
|------|------|------|
| Left | 40px | 增益标签（-48, -36, ..., +48 dB） |
| Bottom | 25px | 频率标签（20, 50, 100, ..., 20k） |
| Top | 10px | 留白 |
| Right | 10px | 留白 |

### 内口区域（Inner Chart Area）

内口 = 外口 - 边距

```
innerRect = QRectF(marginLeft,
                   marginTop,
                   widgetWidth - marginLeft - marginRight,
                   widgetHeight - marginTop - marginBottom)
```

## 影响范围

### 1. CoordinateMapper

- `setViewport()` 接收的应为**内口区域**（不再是全 Widget 区域）
- `freqToX()` / `gainToY()` 等所有坐标映射基于内口像素坐标
- 不需要改动 CoordinateMapper 自身代码，只需调用方传入正确的 viewport

### 2. ViewEqualizer

- `resizeEvent()` 中计算内口 rect 并传给 CoordinateMapper
- Band Handle 位置基于内口坐标
- LPF/HPF Handle 位置基于内口坐标

### 3. VulkanRenderer

- 网格线绘制在内口区域
- 曲线绘制在内口区域
- Band 圆形绘制在内口区域
- **文字刻度标签绘制在边距区域**（margin 内）
- **刻度短线绘制在内口边缘**（网格线顶点扩展到边缘）
- 渲染的 viewport 使用外口尺寸（全 Widget），但网格/曲线坐标基于内口

### 4. VulkanSwapchain

- 无需改动，swapchain extent 仍为全 Widget 尺寸

## SubStage 10A：实现回字形布局

### 10A-1：定义边距常量

在 `ViewEqualizer.h` 或 `VulkanRenderer.h` 中定义：

```cpp
static constexpr float MARGIN_LEFT   = 40.0f;
static constexpr float MARGIN_RIGHT  = 10.0f;
static constexpr float MARGIN_TOP    = 10.0f;
static constexpr float MARGIN_BOTTOM = 25.0f;
```

### 10A-2：修改 CoordinateMapper viewport

在 `ViewEqualizer::resizeEvent()` 中：

```cpp
QRectF innerRect(MARGIN_LEFT, MARGIN_TOP,
                 width() - MARGIN_LEFT - MARGIN_RIGHT,
                 height() - MARGIN_TOP - MARGIN_BOTTOM);
m_mapper->setViewport(innerRect);
```

### 10A-3：修改 VulkanRenderer 网格线

网格线顶点从内口左上角延伸到内口右下角：
- 横线：从 `MARGIN_LEFT` 到 `width() - MARGIN_RIGHT` 的像素 X
- 竖线：从 `MARGIN_TOP` 到 `height() - MARGIN_BOTTOM` 的像素 Y

### 10A-4：修改文字标签位置

- **频率标签**：位于 `marginBottom` 区域内，Y 坐标在外口底部附近
- **增益标签**：位于 `marginLeft` 区域内，X 坐标在外口左侧附近
- **刻度短线**：位于内口边缘，指向内口内部或外部

### 10A-5：修改 Band Handle 位置

Handle 位置基于内口坐标（已经由 CoordinateMapper 处理，无需额外修改）。
