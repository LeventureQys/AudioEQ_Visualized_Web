## 背景与目标

Stage4 完成了渲染器。Stage5 实现最后两层：
- **5A**：`ViewEqualizer`——Qt Widget 内部组合容器，把 Vulkan 渲染区和交互 Handle 组合起来，管理布局和事件分发
- **5B**：`AudioEQ`——对外唯一入口，暴露所有公开 API，连接 Model + Engine + Renderer

完成后，`AudioEQ` 作为一个可用的 `QWidget` 可以被嵌入任意 Qt 应用。

## SubStage 5A：ViewEqualizer

### 目标
`ViewEqualizer` 是 `AudioEQ` 内部的唯一 Widget 子视图，负责：
1. 将 `VulkanWindow`（Vulkan 渲染区）嵌入自身
2. 在 Vulkan 渲染区上方叠加 `BandHandle × N` 和 `LpfHandle` + `HpfHandle`
3. 监听 `EqualizerModel` 的信号，更新 Handle 位置
4. 将 Handle 的鼠标信号向上转发给 `AudioEQ`

### Layout 结构

```
ViewEqualizer (QWidget)
  ├── m_vulkanContainer (QWidget, 由 createWindowContainer 包裹 VulkanQtWindow，全尺寸)
  │   └── VulkanQtWindow (来自 Stage4-4A，承载 VkSurfaceKHR)
  ├── BandHandle[0..N-1] (QWidget, 绝对定位，覆盖在 m_vulkanContainer 上方)
  ├── LpfHandle
  └── HpfHandle
```

`m_vulkanContainer = QWidget::createWindowContainer(m_window, this)`，作为子 Widget 占满 ViewEqualizer。  
使用 `QWidget::raise()` 确保 Handle 层级在 `m_vulkanContainer` 上方。Handle 使用绝对定位（不使用 layout manager），位置由 `updateBandHandlePosition()` 动态设置。

### ViewEqualizer 内部初始化序列

按 Stage4 规定的时序在 `setRenderer()` 完成后立即执行：

```cpp
void ViewEqualizer::setRenderer(VulkanRenderer* renderer) {
    m_renderer = renderer;
    // 创建 Vulkan 窗口（来自 Stage4 的 VulkanQtWindow）
    m_window = new VulkanQtWindow(m_ctx, m_renderer);  // m_ctx 通过另一个 setter 注入或由调用者保证
    m_window->setVulkanInstance(m_ctx->qtInstance());
    m_window->setSurfaceType(QSurface::VulkanSurface);
    m_window->create();
    VkSurfaceKHR surface = QVulkanInstance::surfaceForWindow(m_window);
    m_renderer->initialize(surface, m_window->size());
    m_vulkanContainer = QWidget::createWindowContainer(m_window, this);
    m_vulkanContainer->setGeometry(0, 0, width(), height());
    m_vulkanContainer->lower();  // 放到最底层，Handle 在其上
}
```

**注意**：`ViewEqualizer` 需要新增 `setVulkanContext(VulkanContext*)` 接口（C5-1 契约的补充），由 AudioEQ 在调用 `setRenderer` 之前注入。

### Handle 位置计算

ViewEqualizer 连接 `EqualizerModel::bandChanged(int)` 信号：
```cpp
// 收到 bandChanged(index) 时：
EQBand b = m_model->band(index);
QPointF pixel = m_mapper->toPixel(b.freqHz, b.gainDb);
m_bandHandles[index]->setCenter(pixel.toPoint());
m_bandHandles[index]->move(
    pixel.x() - HANDLE_RADIUS, pixel.y() - HANDLE_RADIUS
);
m_bandHandles[index]->resize(HANDLE_RADIUS * 2, HANDLE_RADIUS * 2);
```

`HANDLE_RADIUS = 20`（像素）。

LPF/HPF 手柄 Y 坐标固定为 `m_mapper->gainToY(0.0)`（0dB 线）。

### resize 处理

```cpp
void ViewEqualizer::resizeEvent(QResizeEvent* event) {
    m_mapper->setViewport(QRectF(0, 0, width(), height()));
    m_vulkanContainer->resize(size());  // VulkanWindow 容器跟随
    // 重新定位所有 Handle
    for (int i = 0; i < m_model->bandCount(); i++)
        updateBandHandlePosition(i, ...);
    // LPF/HPF Handle 位置也更新
}
```

### 禁止事项
- 不直接操作 `VulkanRenderer` 的渲染数据（那是 AudioEQ 的职责）
- 不解析 deltaX/deltaY 为频率/增益（那是 AudioEQ 的职责）

### 验收标准
- `VulkanWindow` 成功嵌入，渲染区可见
- Band Handle 在 `bandChanged` 信号后位置更新
- `resize` 后 VulkanWindow 跟随变化，Handle 位置重新计算

### 构造时初始化序列

```cpp
AudioEQ::AudioEQ(QWidget* parent) : QWidget(parent) {
    // 0. 创建并初始化 VulkanContext（先于 Renderer）
    m_ctx = new VulkanContext();
    if (!m_ctx->initialize()) {
        qCritical("AudioEQ: VulkanContext init failed.");
        // 不抛异常，但后续 m_renderer 不创建，Widget 处于"不可用"状态
        return;
    }

    // 1. 创建 Model
    m_model = new EqualizerModel(this);

    // 2. 创建 CoordinateMapper
    m_mapper = new CoordinateMapper();

    // 3. 创建 CurveEngine
    m_engine = new CurveEngine(this);

    // 4. 创建 VulkanRenderer（不含 Qt 集成）
    m_renderer = new VulkanRenderer(m_ctx, this);

    // 5. 创建 ViewEqualizer（含 Qt-Vulkan 集成）
    m_view = new ViewEqualizer(this);
    m_view->setRenderer(m_renderer);
    m_view->setCoordinateMapper(m_mapper);
    m_view->setModel(m_model);

    // 6. 布局：ViewEqualizer 填充整个 AudioEQ
    auto* layout = new QVBoxLayout(this);
    layout->setContentsMargins(0, 0, 0, 0);
    layout->addWidget(m_view);

    // 7. 连接所有信号槽
    // Model → AudioEQ
    connect(m_model, &EqualizerModel::bandChanged, this, &AudioEQ::onBandChanged);
    // ... (所有信号)

    // ViewEqualizer → AudioEQ
    connect(m_view, &ViewEqualizer::bandDragged, this, &AudioEQ::onBandDragged);
    // ...

    // CurveEngine → AudioEQ
    connect(m_engine, &CurveEngine::totalCurveReady, this, &AudioEQ::onTotalCurveReady);
    // ...

    // 8. 默认 5 个 Band
    setBandCount(5);
}
```

### 拖拽逻辑（onBandDragged）

```cpp
void AudioEQ::onBandDragged(int index, double deltaX, double deltaY) {
    // deltaX/deltaY 是像素增量
    // 当前 band 位置对应像素坐标
    EQBand b = m_model->band(index);
    QPointF curPx = m_mapper->toPixel(b.freqHz, b.gainDb);

    // 加上 delta
    double newX = curPx.x() + deltaX;
    double newY = curPx.y() + deltaY;

    // 反映射回频率/增益
    double newFreq = m_mapper->xToFreq(newX);
    double newGain = m_mapper->yToGain(newY);

    // clamp 到合法范围
    newFreq = qBound(m_mapper->freqMin(), newFreq, m_mapper->freqMax());
    newGain = qBound(m_mapper->gainMin(), newGain, m_mapper->gainMax());

    m_model->setBandFrequency(index, newFreq);
    m_model->setBandGain(index, newGain);
    // EqualizerModel 的 bandChanged 信号会触发曲线重算和渲染更新
}
```

### `setBandCount(int count)` 实现

```cpp
ResultCode AudioEQ::setBandCount(int count) {
    if (count < 0) return ResultCode::InvalidParameter;

    int current = m_model->bandCount();
    if (count > current) {
        // 对数均匀分布分配默认频率
        // freq[i] = 20 * (20000/20)^(i/(count-1))，若 count==1 则 freq=1000
        for (int i = current; i < count; i++) {
            EQBand b;
            b.freqHz = (count > 1)
                ? 20.0 * std::pow(20000.0/20.0, double(i)/(count-1))
                : 1000.0;
            m_model->addBand(b);
        }
    } else {
        while (m_model->bandCount() > count)
            m_model->removeBand(m_model->bandCount() - 1);
    }
    return ResultCode::OK;
}
```

### 禁止事项
- 不直接持有 `VulkanContext`（由 `VulkanRenderer` 内部管理，或通过 `VulkanQtIntegration` 持有）
- 不在 `AudioEQ` 中直接操作 Vulkan API

### 验收标准
- `AudioEQ` 可以嵌入 Qt 应用，显示网格和 5 条曲线
- 拖拽 BandHandle 后曲线实时更新
- `setSampleRate(SR_48000)` 后 X 轴上限更新至 24kHz
- `setLpfEnabled(true)` 后 LPF 手柄可见，曲线更新
- `setBandCount(3)` 减少到 3 个 band 后 Handle 数量同步
