### 测试 8-3：Vulkan Validation 检查

运行 SimpleDemo 的整个交互流程（上述 14 项操作），全程开启 Vulkan Validation Layer。  
检查标准：**0 个 VK_ERROR 或 validation error 输出**（warning 级别允许，需记录）。

验证方法：在 Debug 构建下运行，观察 Qt debug 输出中无 `VUID-` 前缀的错误信息。
