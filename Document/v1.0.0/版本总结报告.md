## 已知限制

1. **macOS**：仅保证编译通过，真机 Vulkan 渲染验证推迟到后续版本（Q14 决策）
2. **SPIR-V Shader**：需要 Vulkan SDK (glslc) 预编译，构建机不要求安装 SDK
3. **字体渲染**：VulkanFontAtlas 依赖 CMake EmbedFont.cmake 生成字节数组（Stage5 已集成）
