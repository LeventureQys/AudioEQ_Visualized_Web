## 背景与目标

Stage3 完成了所有 Vulkan 子模块（Swapchain、Pipeline、BufferPool、FontAtlas、FrameSync）。Stage4 的唯一任务是将它们整合为 `VulkanRenderer`：一个完整的渲染器，能够接受数据（曲线点集、参数），输出一帧完整的 EQ 可视化画面。

完成后，能够渲染出：
- 频率/增益网格线（含坐标轴标注文字）
- 总曲线（绿色宽线）
- 聚焦 Band 的单条曲线 + 填充渐变
- Band 圆形手柄（渲染层，交互层是 BandHandle Widget）
- LPF/HPF 椭圆手柄（渲染层）

## SubStage 4A：VulkanRenderer + VulkanQtIntegration

### 接口定义

```cpp
// src/vulkan/VulkanRenderer.h
#pragma once
#include <QObject>
#include <QVector>
#include <QPointF>
#include <QColor>
#include "volk.h"   // 不要 include <vulkan/vulkan.h>
#include "VulkanContext.h"
#include "VulkanSwapchain.h"
#include "VulkanFrameSync.h"
#include "VulkanPipeline.h"
#include "VulkanBufferPool.h"
#include "VulkanFontAtlas.h"
#include "../AudioEQTypes.h"
#include "../CoordinateMapper.h"

// 每个 Band 渲染所需的数据快照
struct BandRenderData {
    double  freqHz;
    double  gainDb;
    int     index;
    bool    focused;    // true = 绘制高光圆圈
    bool    bypass;
};

struct LpfRenderData { double freqHz; bool enabled; };
struct HpfRenderData { double freqHz; bool enabled; };

class VulkanRenderer : public QObject {
    Q_OBJECT
public:
    explicit VulkanRenderer(VulkanContext* ctx, QObject* parent = nullptr);
    ~VulkanRenderer();

    // 初始化（surface 由 VulkanQtIntegration 提供）
    bool initialize(VkSurfaceKHR surface, QSize size);
    void destroy();

    // 窗口大小变化
    void resize(QSize newSize);

    // 更新渲染数据（主线程调用）
    void setTotalCurve(const QVector<QPointF>& points);   // x∈[0,1], y=dB
    void setBandCurve(int index, const QVector<QPointF>& points);
    void setBands(const QVector<BandRenderData>& bands);
    void setLpf(const LpfRenderData& lpf);
    void setHpf(const HpfRenderData& hpf);
    void setCoordinateMapper(const CoordinateMapper* mapper);  // 用于 dB→y 坐标转换

    // 颜色配置
    void setCurveColor(QColor color);
    void setBackgroundColor(QColor color);

    // 渲染一帧（在 Qt 的 paint 时机调用）
    void renderFrame();

private:
    // 内部：将曲线点集转为 TriangleStrip 顶点（Curve shader 格式）
    // NDC 坐标：x = point.x * 2 - 1, y = -(gainToY(point.y) / height * 2 - 1)
    // lineWidth: 像素宽度（主曲线 2px，单 Band 1.5px）
    QVector<float> buildCurveVBO(const QVector<QPointF>& points, float lineWidthPx);

    // 内部：构建网格线 VBO（水平：增益刻度，垂直：频率刻度）
    QVector<float> buildGridVBO();

    // 内部：Command Buffer 录制
    void recordCommandBuffer(VkCommandBuffer cmd, uint32_t imageIndex);

    VulkanContext*    m_ctx;
    VulkanSwapchain   m_swapchain;
    VulkanFrameSync   m_frameSync;
    VulkanPipeline    m_pipeline;
    VulkanBufferPool  m_bufferPool;
    VulkanFontAtlas   m_fontAtlas;

    // 渲染数据
    QVector<QPointF>           m_totalCurve;
    QMap<int, QVector<QPointF>> m_bandCurves;
    QVector<BandRenderData>    m_bands;
    LpfRenderData              m_lpf;
    HpfRenderData              m_hpf;
    const CoordinateMapper*    m_mapper = nullptr;

    // VBO 缓冲（每帧更新）
    BufferAllocation m_gridVBO;
    BufferAllocation m_totalCurveVBO;
    QMap<int, BufferAllocation> m_bandCurveVBOs;

    // 颜色
    QColor m_curveColor      = QColor(0, 255, 0);     // #00FF00
    QColor m_backgroundColor = QColor(26, 26, 26);    // #1A1A1A

    VkSurfaceKHR m_surface = VK_NULL_HANDLE;
    QSize        m_size;
    bool         m_initialized = false;
};
```

```cpp
// src/vulkan/VulkanQtIntegration.h
#pragma once
#include <QWindow>
#include <QVulkanInstance>
#include "volk.h"   // 不要 include <vulkan/vulkan.h>

class VulkanContext;
class VulkanRenderer;

// QWindow 子类：作为 Qt 中持有 VkSurfaceKHR 的窗口，可通过 createWindowContainer 嵌入 QWidget
// 不使用 QVulkanWindow（它会自管理 swapchain，与我们的 VulkanSwapchain 冲突）
class VulkanQtWindow : public QWindow {
    Q_OBJECT
public:
    explicit VulkanQtWindow(VulkanContext* ctx, VulkanRenderer* renderer);
    ~VulkanQtWindow();

    // 获取 VkSurfaceKHR（Qt 在 setVulkanInstance 后由 surfaceHandle() 返回）
    VkSurfaceKHR vkSurface() const;

    // 当宿主 Widget 想要触发一次渲染时调用
    void requestRender();

protected:
    void exposeEvent(QExposeEvent* e) override;       // 首次曝光时初始化 renderer
    void resizeEvent(QResizeEvent* e) override;       // 通知 renderer resize
    bool event(QEvent* e) override;                   // 处理 UpdateRequest

private:
    VulkanContext*  m_ctx;
    VulkanRenderer* m_renderer;
    bool            m_initialized = false;
};
```

**初始化时序约定**（必须严格遵守，否则 surface 创建会失败）：

```cpp
// 在 ViewEqualizer 构造函数中：
1. m_window = new VulkanQtWindow(ctx, renderer);
2. m_window->setVulkanInstance(ctx->qtInstance());   // ① 必须先设置 instance
3. m_window->setSurfaceType(QSurface::VulkanSurface);
4. m_window->create();                               // ② 然后 create() 才会创建 surface
5. VkSurfaceKHR surface = QVulkanInstance::surfaceForWindow(m_window);
6. m_renderer->initialize(surface, m_window->size()); // ③ 用 surface 初始化 renderer
7. m_widgetContainer = QWidget::createWindowContainer(m_window, parentWidget);
```

每帧渲染由 `VulkanQtWindow::event(QEvent::UpdateRequest)` 触发 `m_renderer->renderFrame()`。  
`requestRender()` 内部调用 `requestUpdate()` 提交一次 UpdateRequest 到 Qt 事件循环。

### 渲染流程

每帧 `renderFrame()`：
1. `m_frameSync.waitAndResetFence(currentFrame)`
2. `m_swapchain.acquireNextImage(imageAvailable, &imageIndex)`（若返回 false → `resize()`）
3. 更新所有 UBO（颜色、矩阵）
4. 若曲线数据脏：`buildCurveVBO` 并 `bufferPool.uploadData`
5. 若网格数据脏：`buildGridVBO` 并上传
6. `recordCommandBuffer(cmd, imageIndex)`
7. 提交 Command Buffer，`m_swapchain.present(...)`
8. `m_frameSync.advanceFrame()`

### 坐标转换（NDC）

```
NDC_x = (pixel_x / viewport_width)  * 2.0 - 1.0
NDC_y = (pixel_y / viewport_height) * 2.0 - 1.0   // y 轴向下
```

CoordinateMapper 提供 `freqToX(freq)` → 视口像素 X，`gainToY(gain)` → 视口像素 Y，再转 NDC。

### 网格线规格

- **垂直网格**（频率）：20Hz, 50Hz, 100Hz, 200Hz, 500Hz, 1kHz, 2kHz, 5kHz, 10kHz, 20kHz  
  - 颜色：`rgba(255,255,255,0.15)`（灰色半透明）
  - 标注文字：频率值（如 "1k"），字号 11px，显示在底部
- **水平网格**（增益）：从 `gainMin` 到 `gainMax`，每 12dB 一条  
  - 0dB 线颜色稍亮：`rgba(255,255,255,0.25)`
  - 其他：`rgba(255,255,255,0.12)`
  - 标注文字：dB 值（如 "+12"），字号 11px，显示在左侧

### 禁止事项
- 不持有 `EqualizerModel` 的引用，不直接读取 model 数据（只接收已处理好的渲染数据）
- 不在 `renderFrame()` 中修改 `EqualizerModel`

### 验收标准
- `initialize()` 成功，不报 Vulkan validation 错误
- `renderFrame()` 在有数据的情况下显示：网格 + 至少一条空白（0dB）曲线
- `resize()` 后不崩溃，下一帧正常渲染
- 设置曲线数据后下一帧能看到曲线变化（人工检视）
