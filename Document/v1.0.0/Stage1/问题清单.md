### S1-Q7：SPIR-V 占位符需要替换为真实编译产物

**背景**：本机未安装 Vulkan SDK / `glslc`，SubStage 1D 生成的 `.spv` 文件为空壳占位符（20字节，仅含 SPIR-V magic number），不含实际 Shader 代码。Stage3-3D 加载时会失败。

**问题**：如何获取真实的 SPIR-V 字节码？

- [x] A. 安装 Vulkan SDK，手动运行 glslc 重编译 .spv 并替换 ← **用户选择**
- [ ] B. 仅提交 GLSL 源码 + 编译脚本
- [ ] C. 推迟到 Stage3 处理

> **答复**：A。安装 Vulkan SDK 编译真实 SPIR-V。

**解决**：已安装 Vulkan SDK 1.4.350.0（`C:\VulkanSDK\1.4.350.0\Bin\glslc.exe`），8 个 Shader 全部成功编译为真实 SPIR-V。文件大小 312~1096 字节，magic number 均为 `0x07230203`。
