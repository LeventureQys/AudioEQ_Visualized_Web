## 1. 背景与目标

### 1.1 业务背景

Stage10 完成"回字形"骨架（外口 / 内口 / 边距）。从 [当前实机效果.png](../当前实机效果.png) 看，骨架已经成型，但视觉与 [目标样式.png](../目标样式.png) 差距较大——网格 / 轴线 / 文字 / 配色 / 范围全方位需要打磨。Stage11 是把"骨架"变成"成品"的最后一步。

### 1.2 锁定的设计决策（来自问题清单）

| # | 决策 | 来源 |
|---|---|---|
| D1 | 频率轴最左档保持 `20Hz`（不抬到 50Hz）；`20Hz` 标签**居中对齐**到 `freqToX(20)`，自然向左溢出到 marginLeft | A1 |
| D2 | SDK 单一 `gainRange`（编辑 = 显示），**不**引入"双范围"。默认值改为 `(-18, +12)`，可通过 `AudioEQ::setGainRange` 动态变更 | A2 |
| D3 | 响度标签**不写** `dB`；频率标签**每档**带 `Hz` / `KHz`（K 大写、无空格） | A3 |
| D4 | gain 网格步长 **6 dB**；freq 网格 **9 档**（与频率标签同步） | A4 |
| D5 | 轴线只画**左 + 底**（L 形），右、上不画 | A5 |
| D6 | 配色照搬目标图采样近似值；**曲线颜色**沿用现有 `AudioEQ::setCurveColor` 接口对外，**默认值**从 `#00FF00` 改为 `#F0A030` | A6 |
| D7 | Stage11 同步关闭 Stage10 的 S10-Q1 / Q2 / Q3 | A7 |
| D8 | Stage11 验收通过后打 `v1.0.0` 正式 tag | A8 |

### 1.3 同步关闭 Stage10 遗留（D7）

| Stage10 问题 | Stage11 答案 |
|---|---|
| S10-Q1 边距值是否合适 | **不合适**，Stage11 改为 `LEFT=44 / RIGHT=12 / TOP=14 / BOTTOM=14`（详见 §3.2） |
| S10-Q2 频率标签方向 | 水平方向不变 |
| S10-Q3 网格线是否延伸 | 不延伸，网格止于内口边缘；轴线（仅左 + 底两条）独立绘制 |

### 1.4 范围声明

**做**：渲染层（VulkanRenderer）全部视觉元素重构、CoordinateMapper 默认值微调、AudioEQ 公共接口默认值 `gainRange` 改 `(-18, +12)` 与 `curveColor` 改 `#F0A030`。

**不做**：
- 不动 SDK 公共接口签名（仅改默认值）
- 不动滤波算法 / CurveEngine 计算
- 不动 BandHandle / LpfHandle / HpfHandle 的 Qt Widget 实现（它们已经是目标图风格）
- 不动 SimpleDemo 控件面板
- 不引入新依赖
- 不做主题切换 / 颜色注入 API（除已开放的 `setCurveColor` / `setBackgroundColor`）

```cpp
QVector<float> VulkanRenderer::buildAxisVBO()
{
    QVector<float> verts;
    float h = static_cast<float>(m_size.height());
    float w = static_cast<float>(m_size.width());

    float leftNdc   = toNdcX(MARGIN_LEFT);
    float rightNdc  = toNdcX(w - MARGIN_RIGHT);
    float topNdc    = toNdcY(MARGIN_TOP);
    float bottomNdc = toNdcY(h - MARGIN_BOTTOM);

    auto addLine = [&](float x1, float y1, float x2, float y2) {
        verts.append(x1); verts.append(y1);
        verts.append(x2); verts.append(y2);
    };

    // L 形：只画左轴 + 底轴
    addLine(leftNdc, topNdc,    leftNdc, bottomNdc);    // 左轴
    addLine(leftNdc, bottomNdc, rightNdc, bottomNdc);   // 底轴
    return verts;
}
```

## 5. SubStage 11B：网格步长与档位

### 5.1 目标

- gain 网格步长从 12 dB → 6 dB
- freq 网格从 23 档 → 9 档（与 [§3.2.4 FREQ_TICKS](#324-频率档位-11b11c-共用) 一致）
- 0 dB 横线**独立** VBO，颜色用 `m_gridZeroColor`（加亮）
- 删除刻度短线（[§1.3 D7 / A5](#13-同步关闭-stage10-遗留d7) 决定不画刻度）

### 5.2 修改清单

**文件**：[src/vulkan/VulkanRenderer.cpp:307-366](../../../src/vulkan/VulkanRenderer.cpp#L307-L366)

#### 11B-1 重写 `buildGridVBO`

```cpp
QVector<float> VulkanRenderer::buildGridVBO(int* outVertexCount)
{
    QVector<float> verts;
    if (!m_mapper) { *outVertexCount = 0; return verts; }

    double gMin = m_mapper->gainMin();
    double gMax = m_mapper->gainMax();
    double fMin = m_mapper->freqMin();
    double fMax = m_mapper->freqMax();

    // —— 横向 gain 网格（步长 6 dB；0 dB 跳过，由 buildZeroDbVBO 单独画）——
    constexpr double GAIN_STEP_DB = 6.0;
    // 从 gMin 起遍历到 gMax，对齐到 6 dB 整倍数
    double g0 = std::ceil(gMin / GAIN_STEP_DB) * GAIN_STEP_DB;
    for (double g = g0; g <= gMax + 0.001; g += GAIN_STEP_DB) {
        if (std::abs(g) < 0.001) continue;  // 跳过 0 dB
        float py = static_cast<float>(m_mapper->gainToY(g));
        float ny = toNdcY(py);
        float nxLeft  = toNdcX(MARGIN_LEFT);
        float nxRight = toNdcX(static_cast<float>(m_size.width()) - MARGIN_RIGHT);
        verts.append(nxLeft);  verts.append(ny);
        verts.append(nxRight); verts.append(ny);
    }

    // —— 纵向 freq 网格（9 档）——
    for (const auto& tick : FREQ_TICKS) {
        if (tick.hz < fMin || tick.hz > fMax) continue;
        float px = static_cast<float>(m_mapper->freqToX(tick.hz));
        float nx = toNdcX(px);
        float nyTop    = toNdcY(MARGIN_TOP);
        float nyBottom = toNdcY(static_cast<float>(m_size.height()) - MARGIN_BOTTOM);
        verts.append(nx); verts.append(nyTop);
        verts.append(nx); verts.append(nyBottom);
    }

    // —— 刻度短线：删除（A5 决定）——

    *outVertexCount = verts.size() / 2;
    return verts;
}
```

> **本 Stage 选方案 a**：把 0 dB 横线 vertices **塞进 `buildAxisVBO`** 的输出（11A 落地时 axis 已是 2 条线 4 vertices；11B 再加 0 dB 共 6 vertices）。


>

```cpp
void VulkanRenderer::setInnerScissor(VkCommandBuffer cmd) const {
    VkRect2D s;
    s.offset.x = static_cast<int32_t>(MARGIN_LEFT);
    s.offset.y = static_cast<int32_t>(MARGIN_TOP);
    s.extent.width  = static_cast<uint32_t>(std::max(0, (int)m_size.width()  - (int)MARGIN_LEFT - (int)MARGIN_RIGHT));
    s.extent.height = static_cast<uint32_t>(std::max(0, (int)m_size.height() - (int)MARGIN_TOP  - (int)MARGIN_BOTTOM));
    vkCmdSetScissor(cmd, 0, 1, &s);
}
void VulkanRenderer::setFullScissor(VkCommandBuffer cmd) const {
    VkRect2D s = { {0, 0}, { (uint32_t)m_size.width(), (uint32_t)m_size.height() } };
    vkCmdSetScissor(cmd, 0, 1, &s);
}
```

**`recordCommandBuffer` 改造**：把原来一开始的 `vkCmdSetScissor` 删掉，按以下顺序穿插：

```cpp
// 默认外口
```
