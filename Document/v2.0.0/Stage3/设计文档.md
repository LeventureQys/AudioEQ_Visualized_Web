# Stage 3 设计文档：Canvas 2D 渲染层

## SubStage 详情

### SubStage 3A：CanvasRenderer 主控 + theme.js

**输出文件**：
- `src/render/theme.js`
- `src/render/CanvasRenderer.js`

#### theme.js

#### CanvasRenderer.js
```js
import { Theme } from './theme.js';
import { CoordinateMapper } from '../core/CoordinateMapper.js';
import { GridLayer } from './GridLayer.js';
import { LabelLayer } from './LabelLayer.js';
import { CurveLayer } from './CurveLayer.js';
import { HandleLayer } from './HandleLayer.js';
import { Defaults } from '../core/types.js';

export class CanvasRenderer {
  constructor(canvas, model, theme = Theme) { ... }
  resize(cssWidth, cssHeight) { ... }  // DPR + viewport
  destroy() { ... }  // cleanup
  markDirty() { ... }  // set dirty flag, schedule rAF
  renderNow() { ... }  // sync render (for testing)
  setTotalCurve(points) { ... }  // → CurveLayer
  setSingleBandCurve(bandIndex, points) { ... }
  clearSingleBandCurve() { ... }
  setTheme(partial) { ... }  // merge partial theme
}
```

Key behaviors:
- **Constructor**: Get canvas 2D context, compute viewport from margin, create CoordinateMapper, create all layers, observe DPR changes via matchMedia
- **Viewport**: `{ x: margin.left, y: margin.top, width: canvasCSSWidth - margin.left - margin.right, height: canvasCSSHeight - margin.top - margin.bottom }`
- **DPR**: `canvas.width = cssWidth * dpr`, `canvas.height = cssHeight * dpr`, `ctx.scale(dpr, dpr)`, listen to `matchMedia('resolution')` change
- **Dirty flag**: `markDirty()` sets `_dirty = true` and schedules `requestAnimationFrame(_renderLoop)`. The loop checks dirty, if dirty: clear canvas, set bg color, draw each layer in order, set dirty=false. If not dirty, skip.
- **Layer draw order**: GridLayer → LabelLayer → CurveLayer → HandleLayer
- **Model listeners**: Listen to model events (`band-changed`, `focused-band-changed`, etc.) and call markDirty()

The CanvasRenderer must store references to all layers so tests can access them.

### SubStage 3B：GridLayer

**输出文件**：`src/render/GridLayer.js`

```js
export class GridLayer {
  constructor(mapper, theme) { ... }
  draw(ctx) { ... }
}
```

Draws:
1. Vertical grid lines at FREQ_TICKS positions (from Defaults.FREQ_TICKS: [20,50,100,200,500,1000,2000,5000,10000,20000])
2. Horizontal grid lines every 6dB: [-24,-18,-12,-6,0,6,12]
3. 0dB line highlighted with `theme.zeroDbColor`
4. L-shaped axis (left + bottom) with `theme.axisColor`
5. Grid lines with `theme.gridColor`

### SubStage 3C：LabelLayer

**输出文件**：`src/render/LabelLayer.js`

```js
export class LabelLayer {
  constructor(mapper, theme) { ... }
  draw(ctx) { ... }
}
```

Draws:
1. Freq labels at FREQ_TICKS positions, below bottom axis, center-aligned
   - Format: < 1000 → "20Hz", >= 1000 → "1KHz" (uppercase K, no space)
2. Gain labels at each 6dB position, left of left axis, right-aligned
   - Format: negative "-24", zero "0", positive "12" (no + sign, no "dB" suffix)
3. Font: `theme.labelFontSize`px system-ui,sans-serif
4. Color: `theme.labelColor`

### SubStage 3D：CurveLayer

**输出文件**：`src/render/CurveLayer.js`

```js
export class CurveLayer {
  constructor(mapper, theme) { ... }
  setTotalCurve(points) { ... }    // points: [{freq, gainDb}]
  setSingleBandCurve(bandIdx, points) { ... }
  clearSingleBandCurve() { ... }
  draw(ctx) { ... }
}
```

Drawing:
- Main curve: `ctx.beginPath()` → `mapper.freqToX(p.freq), mapper.gainToY(p.gainDb)` via lineTo → stroke with `theme.curveColor`, lineWidth=2
- Single band curve (when a band is focused): same color but thinner line (1.5px) + semi-transparent fill below curve
- Y-clamp points to viewport bounds

### SubStage 3E：HandleLayer

**输出文件**：`src/render/HandleLayer.js`

```js
export class HandleLayer {
  constructor(mapper, model, theme) { ... }
  draw(ctx) { ... }  // reads model.allBands(), model.getLpf(), model.getHpf()
}
```

Draws:
- Normal band: circle at (freqToX(band.frequency), gainToY(band.gain)), radius `bandRadius`, fill `bandFill`, stroke `bandStroke`, index number centered
- Focused band: fill becomes `bandFocusFill`
- LPF ellipse: horizontal RX=18, RY=10, text "LPF" inside; enabled → stroke = `curveColor`
- HPF ellipse: same, text "HPF" inside
- Z-order: draw normal bands first, then focused band last (on top)

## 跨 SubStage 契约

| 契约 | 提供方 | 消费方 |
|------|--------|--------|
| CoordinateMapper (viewport, freqToX, gainToY) | Stage 1C | 3A, 3B, 3C, 3D, 3E |
| Theme (colors, sizes, margin) | 3A (theme.js) | 3B, 3C, 3D, 3E |
| Canvas 2D context | Browser API | 3A, 3B, 3C, 3D, 3E |
| EqualizerModel (bands, LPF, HPF) | Stage 2B | 3A, 3E |
| CurveEngine output ({freq,gainDb}[]) | Stage 2C | 3D |
| Defaults.FREQ_TICKS | Stage 1B | 3B, 3C |

所有 layer 的构造函数签名统一：`constructor(mapper, theme, ...extra)`，draw 方法签名统一：`draw(ctx)`。
